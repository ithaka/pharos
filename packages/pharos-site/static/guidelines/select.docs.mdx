<PageSection title="Best Practices">
  <p>
    Select is a type of input that is used in forms, where a user is submitting data and chooses one
    option from a list.
  </p>
  <BestPractices
    Do={
      <ul>
        <li>
          Use when you have 6 or more pre-defined mutually exclusive options for the user to choose
          from (use the select component instead of radio buttons)
        </li>
        <li>Have a default selection when possible</li>
        <li>Use "Select" as the default placeholder option if there's no logical default option</li>
        <li>Labels should be concise</li>
      </ul>
    }
    Dont={
      <ul>
        <li>Avoid using a select for navigational related needs, a dropdown menu will suffice</li>
        <li>Avoid long-winded labels for the options, 1-3 words</li>
      </ul>
    }
  />
</PageSection>

<PageSection title="Content Guidelines">
  <PageSection subSectionLevel={1} title="Labels">
    <ul>
      <li>Succinctly write labels, they are no more than 3 words</li>
      <li>Write in sentence case</li>
      <li>Avoid punctuation and articles ("the," "a," "an")</li>
    </ul>
  </PageSection>
  <PageSection subSectionLevel={1} title="Select Options">
    <ul>
      <li>Start with "Select" as a placeholder if there isn't a default option</li>
      <li>Relates to what the option will ultimately do</li>
      <li>List options alphabetically or in a logical order (like frequency of use)</li>
      <li>Written in sentence case</li>
      <li>Avoid punctuation and articles ("the," "a," "an")</li>
    </ul>
  </PageSection>
</PageSection>
