## Examples

```jsx live
<Fragment>
  <PharosButton
    onClick={() => {
      const event = new CustomEvent('pharos-toast-open', {
        detail: {
          content: 'Hi there!',
        },
      });
      document.dispatchEvent(event);
    }}
  >
    Have a toast!
  </PharosButton>
  <PharosToaster></PharosToaster>
</Fragment>
```
